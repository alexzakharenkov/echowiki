# Echo blockchain Wiki

See wiki here - [https://echo-dev.io/](https://echo-dev.io/)

Install dependences:

```bash
pip3 install mkdocs
pip3 install python-markdown-math
pip3 install pymdown-extensions
pip3 install Pygments
pip3 install .
```

Build HTML version:

```bash
pip3 install .
mkdocs serve
```
