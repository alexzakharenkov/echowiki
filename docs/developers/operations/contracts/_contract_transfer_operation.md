# contract_transfer_operation

Virtual operation that indicates internal transfers from contracts.

```javascript
[
  49,{
    "fee": {
      "amount": 0,
      "asset_id": "1.3.0"
    },
    "from": "1.16.0",
    "to": "0.0.0",
    "amount": {
      "amount": 0,
      "asset_id": "1.3.0"
    },
    "extensions": []
  }
]
```
