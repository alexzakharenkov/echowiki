# asset_claim_fees_operation

Used to transfer accumulated fees back to the issuer's balance.

### JSON Example

```json
[
  43,{
    "fee": {
      "amount": 0,
      "asset_id": "1.3.0"
    },
    "issuer": "1.2.0",
    "amount_to_claim": {
      "amount": 0,
      "asset_id": "1.3.0"
    },
    "extensions": []
  }
]
```
